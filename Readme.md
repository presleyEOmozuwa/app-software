# Full-Stack Application (React + Node.js + AWS EKS)

This project is a full-stack application consisting of a React frontend and a Node.js backend.  
Infrastructure is deployed using Terraform, and CI/CD is handled through GitHub Actions.  
The application runs on AWS EKS (Elastic Kubernetes Service) using Docker containers and Kubernetes manifests.
